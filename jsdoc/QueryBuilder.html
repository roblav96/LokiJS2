Class: QueryBuilder

QueryBuilder

This query builder is the root of each query search. The query contains a sub query and parameters for setup scoring and search options.

Possible sub query types are: TermQuery, TermsQuery, FuzzyQuery, WildcardQuery, MatchQuery, MatchAllQuery, PrefixQuery, BoolQuery, ConstantScoreQuery, ExistsQuery

Example
new QueryBuilder()
  .finalScoring(true)
  .useBM25(1.5, 0.5)
  .term("first_name", "albert")
.build();
// The resulting documents:
// contains the first name albert
// are scored and ranked using BM25 with k1=1.5 and b=0.5

useBM25(k1opt, bopt) → {QueryBuilder}

Use Okapi BM25 as scoring model (default).

See also Lucene#MatchAllDocsQuery and Elasticsearch#BM25.

Parameters:
Name Type Attributes Default Description
k1 number <optional>
 1.2

controls how quickly an increase in term frequency results in term-frequency saturation. Lower values result in quicker saturation, and higher values in slower saturation.
b number <optional>
 0.75

controls how much effect field-length normalization should have. A value of 0.0 disables normalization completely, and a value of 1.0 normalizes fully.
