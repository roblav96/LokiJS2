Class: FuzzyQuery

FuzzyQuery

A query which finds documents where the fuzzy term can be transformed into an existing document field term within a given edit distance (Damerauâ€“Levenshtein distance).

The edit distance is the minimum number of an insertion, deletion or substitution of a single character or a transposition of two adjacent characters.

  • To set the maximal allowed edit distance, use FuzzyQuery#fuzziness (default is 2).
  • To set the initial word length, which should ignored for fuzziness, use FuzzyQuery#prefixLength.

See also Lucene#FuzzyQuery and Elasticsearch#FuzzyQuery.

Example
new QueryBuilder()
  .fuzzy("surname", "einsten")
    .fuzziness(3)
    .prefixLength(3)
.build();
// The resulting documents:
// contains the fuzzy surname einstn (like Einstein or Einst but not Eisstein or Insten)
