Class: Tokenizer

Tokenizer

The tokenizer is used to prepare the string content of a document field for the inverted index. Firstly the string gets split into tokens. After that the tokens will be trimmed/stemmed with defined functions from the queue.

  • To change the splitter function, use Tokenizer#setSplitter.
  • To add functions to the queue, use Tokenizer#add, Tokenizer#addBefore and Tokenizer#addAfter.
  • To remove a function from the queue, use Tokenizer#remove.
  • To reset the tokenizer, use Tokenizer#reset.
